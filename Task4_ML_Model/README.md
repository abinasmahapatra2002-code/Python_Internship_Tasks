# Task 4 – Spam Detection using Machine Learning

## 📌 Project Description
This project implements a simple Spam Detection model using Machine Learning.

The model classifies messages as:
- Spam
- Ham (Not Spam)

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📊 Machine Learning Algorithm
- Multinomial Naive Bayes

## 🚀 How It Works
1. Text messages are converted into numerical format using CountVectorizer.
2. Data is split into training and testing sets.
3. Model is trained using Naive Bayes algorithm.
4. Accuracy is calculated.
5. Custom message prediction is supported.

## ▶ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run Jupyter Notebook:
   jupyter notebook

3. Open Spam_Detection.ipynb
