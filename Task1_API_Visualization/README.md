# Weather Forecast Data Visualization

This project uses the OpenWeatherMap REST API to fetch 5-day weather forecast data and visualize temperature trends using Python.

## 📌 Technologies Used
• Python  
• OpenWeatherMap REST API  
• Pandas  
• Matplotlib  

## 📊 Features
• Fetches real-time weather forecast data  
• Parses JSON response  
• Extracts temperature information  
• Displays temperature trend graph  
• Saves graph as PNG image  

## 🎯 Key Learnings
• Working with REST APIs using HTTP requests  
• JSON data parsing and extraction  
• Data manipulation using Pandas  
• Creating data visualizations using Matplotlib  
• Saving graphical output as image files  

## ▶ How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the script:
   python weather_dashboard.py

The graph will be displayed and saved as **weather_forecast.png**
